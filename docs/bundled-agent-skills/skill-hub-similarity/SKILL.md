# Skill Hub · 相似检测

## 何时使用

用户在 Skill Hub 中打开 **相似检测** 标签，需要理解相似对、分数含义，或决定是否与冲突/删除流程衔接时。

## 算法与界面

- 服务端以 **关键词重叠（Jaccard）** 衡量相似度；视图中的 **灵敏度** 滑块即阈值，越高越严格。
- 支持常见同义词归一（如 `小红书` ↔ `XHS` ↔ `RedNote`）。

## 流程建议

1. 先通过 **Skills** 或 **一键扫描** 保证本机索引较新。
2. 在相似检测视图中查看配对结果（控件布局以当前 Web UI 为准）。
3. 对误报或无需处理的情况，结合列表与冲突页自行判断。
